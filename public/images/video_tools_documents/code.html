Installation

Ensure you have Python 3.10+ and CUDA 12.1 installed on your workstation for optimal neural processing.

bash
# Clone the repository
git clone https://github.com/neural-vfx/upscaler-pro
cd upscaler-pro
# Create virtual environment
python -m venv venv
source venv/bin/activate
# Install dependencies
pip install -r requirements.txt
warning

System Log: Critical Info

FP16 half-precision is enabled by default. If your GPU does not support Tensor Cores, please use the --precision fp32 flag to avoid memory allocation errors.

02/

Model Configuration

Customize the neural weights and tile sizes in the config.yaml file. For high-res 4K renders, increasing tile overlap is recommended.

yaml
model:
  architecture: "Real-ESRGAN"
  scale: 4
  denoise_strength: 0.5
inference:
  tile_size: 512
  tile_overlap: 32
  batch_size: 1
lightbulb

System Log: Optimization Tip

Setting tile_size: 1024 on GPUs with 24GB+ VRAM significantly increases throughput for batch processing.
